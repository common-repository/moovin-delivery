Plugin para entregas de paquetes con Moovin Costa Rica en Woocommerce.

== Description ==
![alt moovin](https://i1.wp.com/www.moovin.me/wp-content/uploads/2021/01/logotipo.png?fit=250%2C65&ssl=1)

### Moovin es la solución logística que tu negocio necesita.

Recogemos tu mercadería y nos encargamos de todo el proceso de entrega. 
Incluso podemos almacenar, alistar y entregar tus paquetes de forma automática con nuestro nuevo servicio de Fulfillment.
Un servicio “todo en uno” con el que no tendrás que preocuparte por nada más que vender, porque nosotros hacemos el resto. 

Gracias a la integración con nuestro plugin para Woocommerce podrás imprimir las etiquetas y pedir la recogida de forma rápida. Podrás conocer en todo momento el estado del envío y su localización en tiempo real, enviamos tus pedidos siempre con la mejor opción de forma automática.

## Funcionalidades

- Entrega de paquetes con servicio express (2h-4h) 
- Entrega de paquetes con servicio rutas (24h-48h) 
- Map picker con google o here maps
- Tracking de paquetes
- Administrador de ordenes
- Envio de notificaciones 
- Configuración de horario para servicios express
- Multimoneda dolares americano USD - Colon Costarricense CRC
- Consulta de tipo de cambio a API Hacienda y BCCR Costa Rica
- Modo extendido o simple para formulario checkout
- Switch para autoconfiguración de zonas moovin 
- Switch para ventas fuera de zona moovin
- Configuración de  métodos de pago que requieran de una confirmación explicita para solicitar la recolección.
- Compatibilidad con cupones y envíos gratis por determinado montos
- Personalización de nombres de metodo de envío

Optimiza tus procesos de envío y reduzca el tiempo de preparación de envíos al integrar en su negocio el servicio logístico de Moovin.

== Frequently Asked Questions ==

= ¿Como funciona el plugin? =

El plugin se conecta mediante una API a Moovin, cuando un cliente ingresa a tu tienda y realiza una compra, puedes solicitarnos la recolección y un mensajero ira por el paquete y lo entregara al cliente.

= ¿Que datos necesito para usar el plugin? =

Los datos de acceso los puedes adquirir fácilmente en nuestro [sitio web](https://www.moovin.me/contacto/) o contactamos por medio del correo electrónico a [ecommerce@moovin.me](mailto:ecommerce@moovin.me).

= ¿Como configuro mi plugin? =

El plugin habilitar en el menu lateral de tu sitio administrador de Wordpress un menu, toca la opción de ajustes y nosotros te damos un paso a paso de las configuraciones que necesitas.
